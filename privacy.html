Privacy Policy

Last updated: May 5, 2026

Overview

TemplateOps Draft Manager is an internal seller tool for preparing Etsy draft listings. It is used only by the shop owner or authorized colleagues.

Data Used

The app may access shop identifiers, listing metadata, draft listing content, listing images and digital product files required to prepare and verify Etsy draft listings.

Data Not Used

The app does not sell personal data, run advertising profiles, process payments or request buyer order data for the current workflow.

Storage

OAuth tokens and listing draft data are stored only in the operator's local working environment or private project storage. Access is limited to the operator of the Etsy shop.

Sharing

Data is not shared with third parties except where required to use Etsy's API and operate the connected Etsy shop workflow.

Contact

For questions, contact the owner of the Etsy developer account associated with this application.
